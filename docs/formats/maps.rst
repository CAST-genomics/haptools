Maps
====

* ``chr`` - chromosome of coordinate (1-22, X)  
* ``var`` - variant identifier   
* ``pos cM`` - Position in centimorgans   
* ``pos bp`` - Base pair coordinate  

.. code-block::python

  {chr}\t{var}\t{pos cM}\t{pos bp}
